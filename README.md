## PROJECT 1

### TO DO LIST APP
This Todo Apps currently only have these feature:
- [CREATE] Add new todo 
- [DELETE] Remove current todo 
- [DETAIL] Check list of your todo. 
you can add up to 8 task, if you want to add more, you have to delete your previous created task
